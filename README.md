# Creating-a-synthetic-instructional-dataset-LLM-s-ability-to-generate-different-types-of-tasks
Проект посвящен исследованию способности LLM генерировать учебные задания разного типа. В этом репозитории также находятся фильтры, на которых происходила автоматическая валидация, затем это использовалось для анализа на языке R, файл в R markdown также находится в репозитории.
