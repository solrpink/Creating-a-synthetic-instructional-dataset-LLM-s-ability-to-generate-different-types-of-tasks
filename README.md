# Creating-a-synthetic-instructional-dataset-LLM-s-ability-to-generate-different-types-of-tasks
Проект посвящен исследованию способности LLM генерировать учебные задания разного типа. В этом репозитории также находятся основные промпты, с помощью которых происходила генерация датасета. Также здесь включено описание датасета, а также файл в формате R, демонстрирующий основной анализ заданий разного типа с помощью фильтров.
